# Funções em JavaScript

## Objetivo
Praticar o uso de funções em JavaScript, incluindo:
- Funções nomeadas  
- Funções anônimas  
- Arrow functions  
- Uso de callbacks  
- Manipulação de arrays  

---
### Operações Matemáticas
- **soma(a, b)** → retorna a soma dos dois números  
- **subtracao(a, b)** → retorna a diferença  
- **multiplicacao(a, b)** → retorna o produto  
- **divisao(a, b)** → retorna o quociente (verifica divisão por zero)

### Manipulação de Arrays
- **filtrarPares(numeros)** → retorna apenas os números pares (arrow function)  
- **calcularMedia(numeros)** → calcula a média dos valores (função anônima)

### Callback
- **executarOperacao(a, b, operacao)** → recebe uma função como parâmetro (callback) e executa a operação desejada.  
  Exemplo: `executarOperacao(8, 2, soma)`

---

## Como Executar
1. Abra o arquivo `index.html` no navegador.  
2. Pressione **F12 → Console** para visualizar os resultados das funções.
